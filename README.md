# VseDoma — недвижимость Дальнего Востока

Платформа для поиска, публикации и подбора недвижимости с геопоиском, real-time чатами и PWA-приложением. Проект был запущен в продакшен, привлекал органический трафик и работал с агентствами.

![Главная страница](screenshots/main.png)

## Стек

| Слой | Технологии |
|------|------------|
| Бэкенд | Django, Django REST Framework, Django Channels |
| Фронтенд | React, TypeScript, Tailwind CSS |
| Карты | Яндекс.Карты API |
| Базы данных | PostgreSQL, Redis |
| Поиск | Elasticsearch |
| Хранилище | MinIO (S3) |
| Мобилки | PWA (Service Workers, push-уведомления) |
| Деплой | Docker, Ansible, GitHub Actions, Nginx |
| Хостинг | VPS TimeWeb, TimeWeb CDN |

## Что умеет

### Поиск и карта
- Полнотекстовый поиск через Elasticsearch
- Фильтры: цена, площадь, тип недвижимости, район
- Просмотр объектов на карте с кластеризацией

![Карта с объектами](screenshots/map.png)

### Карточка объекта
- Фотогалерея (MinIO)
- Характеристики и описание
- Контакты продавца
- Кнопка «Написать» → чат

![Карточка объекта](screenshots/listing.png)

### Чат
- Real-time на WebSockets
- Django Channels + Redis как брокер
- История сообщений в PostgreSQL
- Используется для связи покупателя с продавцом/агентством

### Личный кабинет
- Роли: покупатель, продавец, агентство
- Избранное, история просмотров
- Управление объявлениями
- Push-уведомления о новых сообщениях

### PWA
- Установка на домашний экран
- Офлайн-заглушка
- Push-уведомления
- Геолокация для поиска рядом

### Монетизация
- Платные пакеты размещения для агентств
- Платное продвижение объявлений
- Услуга «Подбор квартиры»

### Админ-панель
- Стандартная Django Admin
- Модерация объявлений и пользователей

### Продакшен-деплой

CI/CD построен на GitHub Actions:
1. Пуш в `main`
2. Линтинг бэка и фронта
3. Сборка Docker-образов
4. Ansible → сервер → `docker-compose down && docker-compose up -d`

## Статус

Проект больше не активен. Код выложен как портфолио. Парсинг на старте помог набрать базу, дальше пошли органические объявления и договоры с агентствами. Фарпост оказался сильным конкурентом, но опыт получен колоссальный.
